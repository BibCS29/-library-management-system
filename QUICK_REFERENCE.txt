COMMON ISSUES & FIXES
════════════════════════════════════════════════════════════════════════════════

ISSUE: "Port 5000 already in use"
FIX:   taskkill /IM python.exe /F
       Then: python app.py

ISSUE: "ModuleNotFoundError: No module named 'flask'"
FIX:   setup.bat (run again)

ISSUE: "Cannot find file .env"
FIX:   In File Explorer: View → Show hidden files (Ctrl+H)

ISSUE: "Cannot connect to localhost:5000"
FIX:   Make sure command window shows "Running on http://127.0.0.1:5000"

ISSUE: "Database is locked"
FIX:   Close all browser tabs, or:
       rmdir /s flask_session
       Then restart server
