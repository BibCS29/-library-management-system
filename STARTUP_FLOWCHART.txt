QUICK REFERENCE (TL;DR)
════════════════════════════════════════════════════════════════════════════════

Step 1: Open Command Prompt (in project folder)
        └─> Right-click in folder → type 'cmd' in address bar → Enter

Step 2: Run Setup
        └─> Type: setup.bat
        └─> Wait 2-5 minutes for dependencies to install

Step 3: Configure .env
        └─> Edit .env file
        └─> Change SECRET_KEY to a random string (32+ chars)

Step 4: Start Server
        └─> Type: python app.py
        └─> See: "Running on http://127.0.0.1:5000"

Step 5: Open Browser
        └─> Type: localhost:5000
        └─> See: Login Page

Step 6: Register & Login
        └─> Create Admin account → Login
        └─> Create Student account → Test features

TROUBLESHOOTING QUICK GUIDE
═══════════════════════════════════════════════════════════════════════════════════

PROBLEM                 SOLUTION
─────────────────────────────────────────────────────────────────────────────────
Port 5000 in use        taskkill /IM python.exe /F
Module not found        setup.bat (run again)
.env not found          Check hidden files (Ctrl+H in Explorer)
Cannot connect          Check server is running in cmd window
Database locked         rmdir /s flask_session
"No such file"          Make sure you're in the project folder
